O presente trabalho consiste na implementação de um jogo de damas digital, com foco no desenvolvimento de uma inteligência artificial (IA) capaz de jogar contra os usuários. Este projeto foi realizado como parte da disciplina de Fundamentos da Inteligência Artificial, ministrada pelo professor Paulo Marcondes Bousfield, na Universidade Univille. Os alunos e autores deste trabalho são Felipe Augusto Soares, Felipe Serrano e Eduardo Busemayer.

2. Requisitos

Requisitos Funcionais:

    Implementação das regras do jogo de damas.
    Desenvolvimento de uma inteligência artificial capaz de jogar contra os usuários.
    Interface de usuário intuitiva para permitir a interação com o jogo.

Requisitos Não Funcionais:

    Responsividade e fluidez no desempenho do jogo.
    Facilidade de uso, especialmente para usuários não familiarizados com o jogo de damas.
    Utilização de tecnologias web para garantir a portabilidade do jogo.

3. Arquitetura do Software

O jogo é composto por três principais componentes:

    Dama: Representa uma peça do jogo de damas.
    Tabuleiro: Representa o estado atual do tabuleiro do jogo.
    Jogo: Controla a lógica do jogo, incluindo as regras de movimento, captura e interação com a inteligência artificial.

4. Instruções de Instalação

Para acessar e jogar o jogo de damas, os usuários podem simplesmente visitar o link para o jogo online e seguir as instruções na tela.
5. Guia do Usuário

    Ao iniciar o jogo, os usuários podem escolher jogar contra outro jogador humano ou contra a inteligência artificial.
    Para mover uma peça, basta selecioná-la e, em seguida, escolher a posição desejada no tabuleiro.
    O jogo informará quando uma peça estiver em posição de captura.
    A partida termina quando um dos jogadores captura todas as peças do oponente ou quando não há mais movimentos possíveis.

6. Manual de Desenvolvedor

O código-fonte do jogo está disponível no repositório do GitHub, onde os desenvolvedores podem cloná-lo, fazer alterações e contribuir conforme necessário.
7. Testes

O jogo possui uma suite de testes automatizados, incluindo testes unitários, de integração e de sistema, para garantir o correto funcionamento de todas as funcionalidades.
